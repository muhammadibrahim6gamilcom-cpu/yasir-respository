Introduction to HTML Forms

Forms are an essential part of websites. They allow users to submit information like name, email, messages, or orders. Learning HTML forms is critical for beginners in web development.

Basic Form Structure

A form starts with the <form> tag and contains input fields and a submit button:

<form action="submit.html" method="post">
  Name: <input type="text" name="name"><br>
  Email: <input type="email" name="email"><br>
  Message: <textarea name="message"></textarea><br>
  <input type="submit" value="Send">
</form>

Input Types

  • text – single line input
  • email – for email addresses
  • password – for password fields
  • checkbox – for multiple options
  • radio – select one option
  • submit – send form data

Styling Forms with CSS

Use CSS to make forms attractive:

input, textarea {
  width: 100%;
  padding: 10px;
  margin-bottom: 10px;
}
input[type="submit"] {
  background-color: #111;
  color: white;
  cursor: pointer;
}

Conclusion

Forms are a key part of interactive websites. Mastering HTML forms along with CSS styling improves your projects and prepares you for backend integration later.
